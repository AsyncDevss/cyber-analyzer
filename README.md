# 🛡️ Gelişmiş Token & Şifre Güvenliği Analizörü

Şifrelerinizin ve Discord bot tokenlerinizin siber güvenlik standartlarına uygunluğunu test eden, kırılma sürelerini hesaplayan tek dosyadan (`index.html`) oluşan cyberpunk temalı bir analiz aracıdır.

## 🚀 Özellikler

* ⚡ **Anlık Güvenlik Skoru:** Yazdığınız şifrenin gücüne göre dinamik olarak renk değiştiren bir ilerleme çubuğu sunar.
* 🕒 **Kırılma Süresi Tahmini:** Şifrenizin kaba kuvvet (brute-force) saldırılarıyla ne kadar sürede çözülebileceğini matematiksel olarak hesaplar.
* 🔑 **Token Doğrulama:** Girilen metnin bir Discord Bot Token formatına (`MT...`) uyup uymadığını akıllı algoritmalarla kontrol eder.
* 🔒 **%100 Güvenli (Gizlilik Odaklı):** Girilen hiçbir veri uzak bir sunucuya veya veri tabanına gönderilmez, tamamen kullanıcının tarayıcısında (client-side) işlenir.

## 🛠️ Kurulum ve Yayınlama

Proje herhangi bir backend veya kütüphane bağımlılığı içermediği için kurulumu çok basittir:

1. Bu depodaki `index.html` kodlarını kopyalayıp kendi oluşturduğunuz bir GitHub deponuza yükleyin.
2. Deponuzun üst menüsünden **Settings (Ayarlar)** kısmına girin.
3. Sol menüden **Pages** seçeneğine tıklayın.
4. *Build and deployment* altındaki *Branch* ayarını `None` yerine **`main`** yapıp **Save** deyin.
5. Birkaç dakika içinde web sitenizin ücretsiz canlı linki en üstte belirecektir!
